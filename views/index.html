FreeCodeCamp API Basejump: URL Shortener

User Stories

  • I can pass a normal web URL and receive a shortened version
  • When I visit the shortened version, I'm redirected to the actual URL

Creating New URL in System

https://plaid-shelf.glitch.me/new/https://www.google.com

Response For Creation

{ "original_url":"https://www.google.com", "short_url":"https://plaid-shelf.glitch.me/8170" }

Visiting Shortened URL

https://plaid-shelf.glitch.me/8170

Will Redirect to Website Saved
